# Algo Lecture Coach

Codex skill for turning BaaaaaaaarkingDog 실전 알고리즘 lecture links, workbook pages, coding-test problems, and user code into practical Java-based study guidance.

## What It Does

- Converts algorithm lecture links into coding-test action plans
- Prioritizes recognition signals, Java implementation patterns, mistakes, and required practice problems
- Supports 10-day sprint planning for the BaaaaaaaarkingDog curriculum
- Reviews Java code for correctness, complexity, counterexamples, and minimal fixes
- Provides memorization-oriented Java syntax and implementation templates

## Structure

```text
SKILL.md
agents/openai.yaml
references/roadmap-10-days.md
references/output-contract.md
references/problem-review-rubric.md
references/java-syntax-patterns.md
```

## Usage

Install or copy this folder into your Codex skills directory, then invoke:

```text
Use $algo-lecture-coach to turn this algorithm lecture link into a concise Java coding-test study plan.
```

## Notes

- Default language: Korean
- Default implementation language: Java
- C++ output is intentionally avoided unless explicitly requested
