# 10-Day Roadmap

Assume the user studies 5-8 hours per day. If the user has less than 3 hours per day, reduce problem count and keep only 필수 problems.

## Day 1

0x00, 0x01, 0x02, basic C++/IO/complexity.

Goal: input/output, arrays, functions, STL basics.

Practice: minimum warm-up problems only.

## Day 2

0x03 Array, 0x04 Linked List.

Goal: indexing, frequency array, erase/insert cost.

Practice: 필수 problems only.

## Day 3

0x05 Stack, 0x06 Queue, 0x07 Deque, 0x08 Parentheses.

Goal: container choice and stack-pattern recognition.

Practice: stack/queue implementation and parentheses.

## Day 4

0x09 BFS.

Goal: grid BFS, distance array, multi-source BFS.

Practice: BFS 필수 problems. This is high priority.

## Day 5

0x0A DFS, 0x0B Recursion.

Goal: traversal, recursion state, base case.

Practice: recursion/DFS basics.

## Day 6

0x0C Backtracking.

Goal: permutations, combinations, pruning, visited.

Practice: N-Queen style and sequence generation.

## Day 7

0x0D Simulation.

Goal: condition-heavy implementation, state update order.

Practice: Samsung-style simulation problems.

## Day 8

0x0E-0x0F Sorting, 0x13 Binary Search if time allows.

Goal: custom sort, lower_bound/upper_bound, parametric search intro.

## Day 9

0x10 DP.

Goal: table definition, transition, initialization, iteration order.

Practice: 1D/2D basic DP.

## Day 10

0x11 Greedy and review.

Goal: greedy proof intuition, counterexample search.

Final task: mixed problem set from BFS, backtracking, simulation, DP, and greedy.

## Exposure-Only Queue

If the user asks for "전체 커버", add these as exposure-only topics:

- 0x12 Math
- 0x14 Two Pointer
- 0x15 Hash
- 0x17 Priority Queue
- 0x18 Graph
- 0x19 Tree
- 0x1B MST
- 0x1C Floyd
- 0x1D Dijkstra

Do not pretend these are mastered in 10 days.
